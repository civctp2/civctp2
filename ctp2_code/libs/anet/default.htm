ActiveNet SDK Packing List

Activision Confidential
Last updated: 11 August 1997

The SDK currently includes the following folders:
  • doc -- SDK Documentation
  • demo -- Demo source and other examples
  • h -- Include files
  • win -- Win32-specific files
    • win/dll -- Network drivers for IPX, TCP/IP, modem, etc.
    • win/lib -- Precompiled libraries
    • win/bin -- Precompiled executables
  • mac.sit -- Archive of Macintosh-specific files. (not present in all builds) Unpacks to:
    • mac/dll -- Network driver for IPX
    • mac/lib -- Precompiled libraries
    • mac/bin -- Precompiled executables
  • dos -- DOS-specific files (not present in all builds)
    • dos/dll -- Network drivers for IPX, TCP/IP, modem, etc.
    • dos/lib -- Precompiled libraries
    • dos/bin -- Precompiled executables
