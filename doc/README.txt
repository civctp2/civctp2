This directory contains the documentation for Call To Power II and the
source code project.  The documentation further categorised as follows:

common/ contains files needed by more than one part of the documentation

dev/    contains documentation for developers working on the source code - it
        contains information on building and deploying CTP2.

user/   contains documentation for users playing the game - includes:
        - the Activision readmes for the v1.1 patch,
        - the readme for the playtest version,
        - the (unfinished) user manual
